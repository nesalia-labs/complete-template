**Notes:**
- 10.5 (per-tenant theming) is the differentiator. No template in the band ships it well.
- 10.10 (skeletons / empty / error states) is a "polished UX" bar — every list page has all three. Half-built features break here first.
- 10.23 (DevTools bubble) is a DX differentiator. It makes the template's internals visible to the buyer during development. The data it surfaces (LLM token usage, API requests, cache hits) also feeds the per-tenant metering story (17.15, 21.7).
- 20.14 (feature flag SDK) is wired into the bubble: active flags are visible at a glance. The bubble is also the fastest feedback loop when a buyer toggles a flag in the admin dashboard (21.13) — real-time via Upstash Realtime.
