# DeesseJS 0.1.0 — Initial public beta

The first public release of DeesseJS is out. The marketing site you're
reading is built on the template itself — same engine, same components,
same patterns you'd ship in your own product.

## What's in 0.1.0

### Added

- **Auth (Better Auth 1.6.19)** — email, OAuth, passkeys, magic links, 2FA
- **Database (Drizzle + Neon)** — type-safe Postgres with migrations + seeds
- **Backend API (Hono + oRPC)** — end-to-end typed RPC without codegen
- **Email (Resend + react-email v6)** — transactional templates + preview app
- **Design system (@workspace/ui)** — shadcn/ui on Base UI, Geist-aligned tokens
- **Documentation (Fumadocs 16)** — full-text search, MDX, code blocks
- **Blog + changelog (content-collections)** — see the [related post](/blog/why-content-collections)

### Notes

This is the **founding release**. Founder pricing ($99, $249, $499) is
capped at 50 founders and closes July 31, 2026.

If you find a bug, open an issue. If an agent breaks the build, that's a
feature request — we want to know.
