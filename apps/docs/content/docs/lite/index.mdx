DeesseJS Lite is the free, open-source counterpart to [Template](/docs). It's a stripped-down, demo-first Next.js scaffold you can clone, run locally, and extend at your own pace — no license, no payments.

## What you get

A minimal but production-shaped Next.js app:

- Next.js 16 + App Router + Tailwind CSS v4
- A single workspace (`apps/lite/`) mirroring the template structure
- shadcn/ui primitives ready to install
- No backend, no auth, no payments — just the front-end skeleton

## How it relates to Template

Think of Lite as the public demo. It shows what DeesseJS looks like in practice and gives you something concrete to fork. When you're ready to ship a real product, [Template](/docs) is the paid version with the full stack wired up: auth, database, API, billing, mail, and a curated design system.
