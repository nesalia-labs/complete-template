The user wants me to **reply in English**, regardless of the language they use to address me.

- They sometimes write in French with a casual register ("chef", "toussa", "yo") but that does NOT mean they want French replies.
- **How to apply:** Default replies to English. If the user explicitly asks for French (or another language) for a specific conversation, switch — but don't infer it from their input language.
